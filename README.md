# Titanic-survivor--prediction


This project predicts whether a passenger survived the Titanic disaster or not . The prediction is based on passenger details such as age, gender,  fare, etc.
This project is made using Jupyter Notebook and Python.

--------
 # Goal of the Project
To predict:
1 → Passenger survived

0→ Passenger did not survive

----
# Dataset Used


The dataset titanic.csv contains:

Passenger class,Gender,Age,Fare etc details

----
 # Tools & Libraries Used

This project is done in Jupyter Notebook using:

Python

Pandas

NumPy


Matplotlib / Seaborn

-----

# What I Did in This Notebook

Loaded the dataset in Jupyter Notebook

Cleaned the data

Handled missing values

Converted text data into numbers

Trained the machine learning model

Predicted survival results

----

# Output
The model predicts whether a passenger:

Survived (1)

Did not survive (0)
